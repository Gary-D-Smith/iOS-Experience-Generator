# iOS Experience Generator

A React app that randomly selects iOS input and output capabilities, then uses Anthropic's Claude to generate creative app experience ideas connecting them.

## Project Structure

- `app/` - Next.js app directory with pages and API routes
- `lib/` - Utility functions for parsing inputs/outputs and loading prompts
- `Inputs.md` - iOS input capabilities
- `Outputs.md` - iOS output capabilities
- `prompt-template.txt` - Customizable prompt template
